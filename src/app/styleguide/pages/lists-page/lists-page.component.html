Dropdown Lists

You can use dropdown lists for any action which requires you to select a single option out of many.

<div class="dropdown"> <button class="btn btn-primary dropdown-toggle" id="dropdownMenuButton" data-toggle="dropdown"> Dropdown Button </button> <div class="dropdown-menu" aria-labelledby="dropdownMenuButton"> <a class="dropdown-item" href="#">Action</a> <a class="dropdown-item" href="#">Another action</a> <a class="dropdown-item" href="#">Something else here</a> </div> </div>

We have two types of toggle-lists, Dropdown and Dropup. They can be used interchangeably by changing the container
class (labeled in green) from dropdown to dropup and vice-versa.
Similarly, we can change the type of button we would like to use for the list by changing the button's class (also labeled in green). To learn more about how you can change between button types click here.
