All form groups can have different layouts and can have our grid integrated easily by adding row and col classes.
To create a input field that should serve as a search bar, just add the search class to the form-group as shown.
You can also add labels to different input elements.

Field Sizing

Fields height and width can be customized as well by adding certain classes.

<form> <div class="form-group"> <input class="form-control form-control-lg" type="text" placeholder=".form-control-lg"> </div> </form> <form> <div class="form-group"> <input class="form-control" type="text" placeholder=".form-control"> </div> </form> <form> <div class="form-group"> <input class="form-control form-control-sm" type="text" placeholder=".form-control-sm"> </div> </form>

You can choose from three predefined sizes for any input field by altering the class name shown in green.

Number Fields

Number input fields can be easily implemented as follows. They are important for when you need to take a number input from the user in a robust way.
